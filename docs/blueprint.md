# **App Name**: PortfolioPro

## Core Features:

- Hero Section: Displays the developer's profile picture, name, title, tagline, and a call-to-action button.
- About Section: Presents the developer's bio, education, and professional highlights.
- Skills Section: Visual representation of the developer's skills using cards or progress bars.
- Services Section: Lists the services offered by the developer with short descriptions and accompanying icons.
- Portfolio Section: Showcases completed projects with project images, descriptions, and links to live demos.
- Contact Section: Displays contact details and includes a contact form for direct messaging.
- Admin Panel: An interface allowing the user to Create, Read, Update, and Delete (CRUD) operations on portfolio data.

## Style Guidelines:

- Background color: Dark theme background (#121212) for a modern feel.
- Primary color: Soft neon blue (#64B5F6) for highlighting and accents.
- Accent color: Subtle purple (#BB86FC) for interactive elements.
- Body and headline font: 'Poppins', a sans-serif font, providing a contemporary, precise feel.
- Consistent, tech-professional icons for skills, services, and contact information.
- Clean sections with ample spacing for a clutter-free experience.
- Smooth hover animations and micro-interactions for enhanced engagement.
